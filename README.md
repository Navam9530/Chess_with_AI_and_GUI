# Chess_with_AI_and_GUI

**Prerequisites:**
* Python3
* GTK (_for Windows only!_)
* Python3 Libraries:
    * python-chess
    * pygame
    * cairosvg

**Note:**
* Run the **`main.py`** file to play the game!
* Choose to play with _PRO AI_ on a _decent spec_ PC only to avoid the errors the OS throws!
